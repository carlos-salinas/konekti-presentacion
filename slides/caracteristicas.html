Características

Estas son algunas de las características más destacables:

  • Independiente del gestor de base de datos
  • Persistencia a través de JPA mediante Eclipselink
  • Desacoplado mediante inyección de Spring
  • Desarrollo de todo el stack mediante JAVA
  • Interfaz modular usando Vaadin Framework
  • Actualizaciones en caliente sin apagar el servidor
